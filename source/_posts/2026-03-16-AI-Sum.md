### 电影报幕

在电影片尾（end credits）中，常见的英文翻译是：

- 导演 → Director
- 编剧 → Writer 或 Screenwriter

更符合电影行业惯例的写法通常是：

```
Director（导演）
Written by（编剧）
```
常见片尾格式示例
```
Director
Zhang San

Written by
Li Si
```
或者：
```
Directed by Zhang San
Written by Li Si
```
如果同一个人既是导演又是编剧

常见写法：
```
Written and Directed by
Zhang San
```
或者：
```
A Film by Zhang San
```
