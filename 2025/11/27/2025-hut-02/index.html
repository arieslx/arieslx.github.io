Day5续上篇

话说昨天忍无可忍给claude冲了会员，今天让chatgpt帮我写了个prompt批量处理我剩余的二十多个场景，gif图先用placeholder代替，claude做的非常不错，它甚至教了我怎么调试。以下是我的prompt：

The current project is a multi-platform framework built with Pixi, React, and Express. I have already created two scenes and a static player. However, I’m experiencing issues with scene positioning: the Pixi scene positions in this project do not match the layout from my previous web-based project, causing misalignment.

I have now copied the the_grand_design project into the current codebase.
Please do NOT modify any code inside the the_grand_design project.

Instead, using the App.js, the LESS files, and the other related files from the_grand_design as references, recreate all 23 scenes defined in App.js from line 389 to line 435 inside the current project.

In the current project, please reference the static files inside public/scenes2. For any GIF images, replace them temporarily with a placeholder image.

Your task is to rebuild these 23 scenes in a Pixi-compatible way while preserving their layout and structure from the original project.

  • 用gemini生成了三张静态png图充作player，目前还没开始做player。
  • 把白底的paler丢进jimeng帮我抠图并导出，没开会员有水印，好吧，我先用着。
  • 两个场景，把placeholder换成三帧的png做gif图。

明天继续，今天早点休息吧。
